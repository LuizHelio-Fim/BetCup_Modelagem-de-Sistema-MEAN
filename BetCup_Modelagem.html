BetCup

Modelagem do Sistema de Apostas da Copa | Angular + Node + MongoDB

Prototipagem visual e documentação do fluxo de apostas, ranking e painel administrativo para um sistema esportivo.

Visão Geral do Sistema

Cadastro/Login de usuário

Usuários criam contas, acessam seu painel e protegem o acesso com autenticação JWT.

Registro de apostas

Apostadores registram palpites para jogos da Copa, visualizam status e acompanham ganhos de pontos.

Ranking de jogadores

Ranking público com posições, pontuações e histórico de apostas para engajar os participantes.

Painel administrativo

Admin monitora usuários, apostas e resultados oficiais, com controles para atualizar partidas.

Gestão de jogos da Copa

Cadastro de partidas, definição de chaves e resultados abruptos para cálculo automático de pontos.

Dashboard de estatísticas

Visão de desempenho com gráficos, totais de apostas e indicadores de engajamento do torneio.

Estrutura de Diretórios

Frontend
  • src/app/pages - telas principais do Angular
  • src/app/components - cards, layouts e widgets
  • src/app/services - consumo de API e lógica de apostas
  • src/app/guards - proteção de rotas e roles
Backend
  • controllers - endpoints de usuários, apostas e jogos
  • routes - rotas REST organizadas por domínio
  • middlewares - JWT, validação e autorização
  • repositories - acesso ao MongoDB
  • config - variáveis de ambiente e conexões

Modelagem Técnica

Arquitetura

O BetCup é concebido como:

  • Angular SPA para interface rica e navegação fluida.
  • API REST Express para rotas de apostas, jogos, usuários e ranking.
  • MongoDB como banco de dados NoSQL para collections de users, games, bets e rankings.
  • JWT Authentication para segurança de sessão e proteção de rotas.
Componentes principais

O protótipo considera:

  • Login / cadastro com validação de formulário e flow de onboarding.
  • Painel de apostas com palpites por partida e histórico.
  • Ranking global com pontuação acumulada.
  • Painel administrativo para gerenciar resultados e jogos.
  • Dashboard de estatísticas para supervisão do torneio.

Fluxo do Sistema

  1. Usuário cria conta no BetCup e define credenciais.
  2. Realiza login e acessa o painel de apostas.
  3. Visualiza jogos da Copa disponíveis para palpitar.
  4. Registra aposta informando placar previsto para cada partida.
  5. Administrador informa resultado oficial das partidas.
  6. Sistema calcula pontuação com base na precisão do palpite.
  7. Ranking é atualizado e os usuários acompanham sua posição.
